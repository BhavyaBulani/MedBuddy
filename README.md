# 🏥 AI Medical Prescription Assistant

An AI-powered web application that simplifies medical prescriptions by extracting, interpreting, and presenting them in a user-friendly format.

## 🚀 Features

- 🧠 AI-based prescription interpretation (including handwritten prescriptions)
- 💊 Simplified medicine descriptions and usage details
- ⏰ Medicine tracking and reminders
- 📍 Nearby pharmacy locator for easy access to medicines
- 🔐 User authentication and profile management

## 🛠 Tech Stack

- **Frontend:** React.js  
- **Backend:** Node.js  
- **Database & Auth:** Supabase  
- **AI Integration:** Gemini API  

## 📌 Problem Statement

Medical prescriptions are often difficult to understand due to complex terminology and handwriting. This project aims to bridge that gap by making prescriptions accessible and easy to interpret for everyone.

## 💡 Solution

This application uses AI to:
- Extract information from prescriptions  
- Convert it into readable and structured data  
- Provide clear explanations of medicines  
- Assist users in managing their medication effectively
